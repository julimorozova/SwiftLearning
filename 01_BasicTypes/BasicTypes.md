## Переменные и константы 

Используются для хранения данных  в приложении 

**let** - обьявляет константу, изменить значение нельзя, значение остается постоянным, например id 
 
**var** - обьявляет переменную, изменить значение можно, занчение изменется в будущем - счетчик, состояние приложения 


``` swift 
let name = "Ivan"
var adress = "Despota street"
```
По умолчанию, обьявляем константу, они быстрее работают и безопаснее, выгоднее с точки зрения памяти, программа выделяет на нее паметь и не следит после обьявления.

## Основные типы данных 

Swift - язык со строгой типизацией
Это позволяет предотвращать ошибки, повышать производительность, улучшать читаемость кода 

### Целые числа 

Int (зависит от платформы, обычно 32 или 64 бита) 
UInt беззнаковое целое значение 


### Числа с плавающей точуой 

Float 32-битное 
Double 64-битное 

по умолчанию объявляется Double

### String
последовательность символлов (текст)

### Character 
отдельный символ 

### Объем в байтах 

**Int, UInt** - 4 байта на 32-битной системе, 8 байт на 64-битной системе

**String** - рахмер динамичен, зависит от содержания текста, не имеет фиксированного размера в байтах

**Bool** обычно занимает 1 байт 

**Double** обычно занимает 8 байт, так как это 64-битное число с плавающей точкой 
**Float** обычно занимает 4 байт, так как это 32-битное число с плавающей точкой 

## Конкатенация строк 

Объединение строк или преобразование чисел в строки для отбражения 
Применяется при:
- создании сообщений от пользователей
- формировани запросов или команд
- логирование информации 

## Интерполляция строк и чисел 

позволяет встравлять значения переменных внутрь строкового литерала 

``` swift 
let name = "Anna"
let message = "User \(name)"
```
Применяется при:
- формирования сообщений для пользователей
- логирование и отладка приложения
- созжание динамических строк на осенове данных 

Преимущества: 
- не нужно явно преобразовывать Int в String 
- код выглядит более чисто и понятно 
 -можно вставлять сложные выражения
